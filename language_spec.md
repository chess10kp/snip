# Language Specifications 

## Syntax 
### Keywords 

- Data Types: `int`, `char`, `string`, `bool` , `float`, `double`, `nothing`, `null`
 
- Keywords: 
`const` `fn` `if`/`elif`/`else` 
`while` `for` `return`
`break`
`class` `this` `new`
`true` `false`


- Arithmetic: `+`, `-`, `*`, `/`
- Comparison: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical: `&&`, `||`, `!`

- Identifier regex: [a-zA-z_]

- Comments:  #
